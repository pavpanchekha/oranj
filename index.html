oranj by pavpanchekha

A Pythonic dynamic programming language

Dependencies

  • Python 2.6 or better
  • elementtree (standard with python)

Install

  1. Download source code
  2. Become root
  3. Run python setup.py install
  4. Run oranj to see if everything worked
