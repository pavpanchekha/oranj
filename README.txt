oranj is a pythonic, dynamic programming language

It is new dynamic, interpreted programming language, with
a focus on GUI and web development. It makes programming
enjoyable and easy, and makes enabling advanced features
direct and fast.

For installation instructions, see the file INSTALL
Visit http://pavpanchekha.github.com/oranj/ for more.

oranj is open-source and community-driven. Please do
whatever you wish to it, within the terms of the GPL.
Please submit bugs at http://github.com/pavpanchekha/oranj/issues.
